Glossário de HTML

Termos e definições importantes

!DOCTYPE

Informa o browser que se trata de um documento HTML
a
Define um hiperlink
dl
Define uma lista de descrição
dt
Define termos ou nomes
dd
Descreve a definição de cada termo ou nome
b
Exibe o texto em negrito

strong

Define texto com forte importância, geralmente exibindo-o em negrito
i
Define uma parte do texto em uma voz ou humor alternativo, geralmente exibido-a em itálico.
em
Enfatiza o texto, geralmente exibindo-o em itálico.

mark

Define o texto que deve ser marcado ou destacado.

small

Define um texto menor.
del
Define o texto que foi excluído de um documento, geralmente riscado com uma linha pelo browser.
ins
Define um texto que foi inserido em um documento, geralmente sublinhado pelo browser.
sub
Define texto subscrito, que aparece meio caractere abaixo da linha normal e, às vezes, é renderizado em uma fonte menor.
sup
Define texto sobrescrito, que aparece meio caractere acima da linha normal e, às vezes, é renderizado em uma fonte menor.

fieldset

Agrupa elementos relacionados em um formulário desenhando uma caixa ao redor.

select

Cria uma list drop-down selecionável pelo usuário.

input

Especifica um campo de entrada onde o usuário pode inserir dados.

textarea

Define um input de múltiplas linhas.

hr

Exibe uma linha ou régua horizontal divindo a página (horizontal rule).
